Overall CPU Analysis¶

Abstract¶

CPU has become a big part of our lives. It is in our phones, desktop, laptop, glasses and even toasters.this report will illustrate how CPUs are improving and where is the future headed by comparing data gathered from two well known websites with over 3000 CPUs analyzed.

Introduction¶

Everyone knows what is a CPU (central processing unit), but what makes CPUs so different from each other. On one hand one might find an expensive CPU that offers low performance, while the other might find a CPU with higher performance but with half the price of the expansive one. No matter how different is the CPU architecture from one another they will always have common variables that are sheared with the consumer. We will be using some of these variables to analysis the datasets acquired. The first keyword we will be using is TDP which stands for Thermal design power which means how much power is consumed when the CPU is on full power and it is measured by watts. The keyword CPU Transistor Size, refers to the size of transistor that are fitted on the CPU itself and it is measured by nm which stands for nanometer. Lastly, the keyword CPU multiplier refers to the speed ratio between the front side bus (FSB) and the CPU.

This report will seek to answer the following questions:

  1. Which CPU has the most efficient power consumed per watts ?
  2. Does price effect performance?
  3. Does the CPU clock speed effect the power usage?
  4. What does multiplier effect in a CPU ?
  5. Is it true that the CPU Transistor Size does not reflect performance ?
  6. Does lower CPU Transistor Size means more power efficiency ?
  7. What can we predict for CPU future ?

#filters
fter = '[^0-9\.]+'
fterABC = '[^A-Za-z]+'
filtter = '[^A-Za-z0-9]+'

#jan 2014 --> 1
def MonthToInt(month):
    date = re.sub(fterABC, ' ', month).lower()
    if 'jan' in date:
        return 1
    elif 'feb' in date:
        return 2
    elif 'mar' in date:
        return 3
    elif 'apr' in date:
        return 4
    elif 'may' in date:
        return 5
    elif 'jun' in date:
        return 6
    elif 'jul' in date:
        return 7
    elif 'aug' in date:
        return 8
    elif 'sep' in date:
        return 9
    elif 'oct' in date:
        return 10
    elif 'nov' in date:
        return 11
    elif 'dec' in date:
        return 12
    else:
        return 0

# Remove a pattren from a list
def RemoveUnwanted(datasample, fpattren):
    count = 0
    for oneRaw in datasample:
        datasample[count]= float(re.sub(fpattren, '', oneRaw))
        count+=1

# Draw Poly line
def curvLine(x, y, dta, poly_num, color='r-'):
    x1 = pd.DataFrame({'x': np.linspace(x.min(), x.max(), len(dta))})
    # 2-nd order polynomial
    if poly_num ==1:
        poly_ = smf.ols(formula='y ~ 1 + x', data=dta).fit()
    elif poly_num == 2:
        poly_ = smf.ols(formula='y ~ 1 + x + I(x ** 2.0)', data=dta).fit()
    elif poly_num == 3:
        poly_ = smf.ols(formula='y ~ 1 + x + I(x ** 2.0) + I(x ** 3.0)', data=dta).fit()
    plt.plot(x1.x, poly_.predict(x1), color, label='Poly n=%d $R^2$=%.2f' % (poly_num,poly_.rsquared),alpha=0.9)
    plt.legend()
# ceil a number to a specific decimal
def roundup(x, rou=20):
    return int(math.ceil(x / rou)) * rou

Conclusion¶

The last graph gives us an idea where is the future of CPUs heading, CPUS is heading toward more power efficient CPUs that make use of smaller transistor size and we can safely say that by 2020 CPUs will be using 5nm transistors or less. The smaller the transistors the more space are available for other transistor or even new technology to be fitted. The smaller the transistors the less power will mobile devices consume .In the end, it is all about the transistor size, so we are relying more on the Nanotechnology.
